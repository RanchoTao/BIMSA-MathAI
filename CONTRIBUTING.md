# 贡献指南

感谢你为 BIMSA 非官方学习知识库做出贡献。

## 可贡献内容
- 课程资料：讲义索引、知识点笔记、作业复盘、考试复盘
- 学习经验：学习方法、时间管理、踩坑记录
- 科研成长：论文阅读、写作模板、研究流程
- 勘误修正：错别字、失效链接、目录错位、命名不一致

## 目录与命名规范
- 目录名统一使用英文 `kebab-case`。
- 文件名统一使用英文，避免空格、中文标点、随意编号。
- 文档标题可使用中文，强调可读性。
- 课程目录统一保留：
  - `README.md`
  - `notes/`
  - `assignments/`
  - `exams/`
  - `resources/`

## 提交流程
1. Fork 并创建分支。
2. 按目录规范新增或修改文件。
3. 若涉及结构调整，请同步更新 `CHANGELOG.md`。
4. 提交 Pull Request，说明：
   - 改动目标
   - 涉及目录
   - 是否迁移了历史资料
   - 是否存在待确认问题

## 内容质量建议
- 尽量给出来源、上下文和适用范围。
- 结论与经验建议配套示例，避免仅给“口号式建议”。
- 对主观性较强内容，建议增加“适用对象/前提条件”。

## 合规与边界
- 请勿上传个人隐私信息。
- 请勿上传受版权或课程平台限制传播的原始资料。
- 作业与考试相关内容建议做匿名化与脱敏处理。
